### ЗАПУСК (с созданием вирутального окружения):
source venv/bin/activate
python3 main.py
-------------------------------------------------
Если возникнут проблемы с venv, можно подгрузить все
необходимые библиотеки при помощи команды:
pip3 install --upgrade -r requirements.txt

Если и после этого не запустится, то очень грустно :(
feedback: berezniker@mail.ru
-------------------------------------------------

[!NOTE] ЗАПУСКАТЬ НУЖНО С АНГЛИЙСКОЙ РАСКЛАДКОЙ
        иначе не будет реакции на клавиатуру

### Интерактивная часть:
W,A,S,D -- движение камеры
мышка -- поворот сцены, как в шутерах
колёсико -- zoom
Q -- on/off каркасная визуализация воды
E -- добавление случайных возмущений (=капель)
R -- on/off режим дождя
B -- on/off skybox
F1, F2, F3 - смена камер
F5 - пауза симуляции

База 30 баллов:
___Реализация выполнена на OpenGL3.x
___Численное решение реализовано на GPU

Дополнительная часть:
___Текстуры на объектах (+1)
___Визуализация возмущающихся объектов - капли (от +1 до +4)
___Управление камерой (от +1 до +2)
___Несколько камер (+1)
___Отражение (+2)
___Окружение надводное: skybox (от +1 до +4)
